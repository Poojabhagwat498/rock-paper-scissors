# ✊✋✌️ Stone Paper Scissors Game

A fun and interactive **Stone Paper Scissors (Rock Paper Scissors) Web Game** built using **HTML, CSS, and JavaScript**.

Play against the computer and test your luck and strategy in this classic game!

---

## 📌 Project Overview

This project is a browser-based game where the user plays against the computer. The computer randomly selects an option, and the winner is decided based on standard game rules.

It demonstrates:
- JavaScript logic building  
- DOM manipulation  
- Event handling  
- Random number generation  
- UI interaction  

---

## ✨ Features

- 🎮 Play against computer  
- 🔀 Random computer moves  
- 🏆 Instant winner result  
- 🔁 Play multiple rounds  
- 📊 Score tracking *(if implemented)*  
- 📱 Simple and responsive UI
